# AWS EC2 Web Server Deployment

## Overview

This project demonstrates deployment of a static website on an AWS EC2 instance using Apache web server.
It shows how cloud-based virtual servers can be used for hosting applications.

---

## Tech Stack

* AWS EC2
* Ubuntu
* Apache
* HTML

---

## What I Did

* Launched and configured an EC2 instance (Ubuntu)
* Set up security groups (HTTP and SSH access)
* Installed and started Apache web server
* Deployed a static website on the server
* Accessed the website using public IP
